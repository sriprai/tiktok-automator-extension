Step 1: Load Extension in Chrome

  1. Open Chrome browser
  2. Navigate to chrome://extensions/
  3. Enable "Developer mode" (toggle in top right)
  4. Click "Load unpacked" button
  5. Select the chrome-extension folder

Step 2: Verify Installation

After loading, you should see:

  • TikTok Video Automator extension in your extensions list
  • Extension icon in Chrome toolbar
  • No error messages in console

Step 3: Test the Extension

Click the extension icon to open the popup. You should see:

  • Login screen (if not logged into web app)
  • Or your videos/products if already logged in
  • Functional tabs and refresh buttons

Important: For authentication to work:

  1. Make sure your web app is running at http://localhost:3000
  2. Log into your web app in a separate tab
  3. The extension will automatically detect your login
  4. If you see 401 errors, refresh the extension popup after logging in

API Integration

The extension connects to your TikTok Video Automator web app at:

const API_BASE_URL = "http://localhost:3000"; // Change for production

Endpoints used:

  • GET /api/auth/me - User authentication
  • GET /api/video-tasks - Video library data
  • GET /api/tiktok/accounts - TikTok accounts

Permissions

Required permissions in manifest.json:

  • storage - Save user data locally
  • activeTab - Interact with current tab
  • scripting - Inject scripts into TikTok
  • cookies - Set TikTok authentication
  • tabs - Open/manage browser tabs
